# Rezka Dual Subtitles (Chrome Extension, MV3)

Расширение для `rezka.si`, которое перехватывает RU/EN субтитры, объединяет их по времени и рисует 2 строки поверх видео.

## Структура

- `manifest.json` — конфигурация MV3.
- `background.js` — дефолтные настройки.
- `content.js` — логика UI, хранение дорожек, рендер поверх видео.
- `modules/subtitle-parser.js` — парсинг SRT/VTT и merge по таймкодам.
- `injected/interceptor.js` — перехват `fetch`/`XMLHttpRequest` в контексте страницы.

## Запуск

1. Откройте `chrome://extensions`.
2. Включите **Developer mode**.
3. Нажмите **Load unpacked** и выберите папку `extension`.
4. Откройте `https://rezka.si/*` с видео и субтитрами.

## Как использовать

1. В плеере выберите русские субтитры — расширение сохранит RU дорожку.
2. Переключите на английские — расширение сохранит EN дорожку.
3. После получения обеих дорожек появятся двойные субтитры.

## Управление в оверлее

- Включить/выключить двойные субтитры.
- Переключить порядок языков (`EN сверху`).
- Изменить размер шрифта.
